# 🔎 Data Leak Detector

A cybersecurity tool that scans text for sensitive leaked data like emails, phone numbers, Aadhar, PAN, and more.

## 🚀 Live Demo
👉 [Click here to try it](https://dikshachauhan.github.io/data-leak-detector)

---

## 📌 About This Project

Data leaks happen when sensitive personal information is accidentally shared or exposed. This tool helps users **scan any text** to detect if it contains sensitive or leaked data.

---

## 🔍 What It Detects

| Type | Description |
|------|-------------|
| 📧 Email Address | Personal email exposed |
| 📱 Phone Number | Indian mobile numbers detected |
| 🪪 Aadhar Card | 12-digit Aadhar number |
| 💳 PAN Card | PAN card format detected |
| 💰 Credit/Debit Card | 16-digit card numbers |
| 🔑 Password | Plain text passwords |
| 🔢 OTP | One-time passwords in text |
| 🏦 Bank Account | Account numbers detected |
| 🏛️ IFSC Code | Bank branch codes |
| 🌐 IP Address | Network information |
| 🎂 Date of Birth | Personal DOB information |
